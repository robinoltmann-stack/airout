# AirOut – Luftqualität & Pollen

**Dein täglicher Outdoor-Ratgeber für Allergiker, Sportler und Familien.**

AirOut zeigt dir auf einen Blick, ob heute ein guter Tag zum Rausgehen ist – basierend auf Luftqualität, Pollenwerten und UV-Index an deinem genauen Standort.

## Features

- **Ampel-Status** – Grün/Gelb/Rot auf Basis aller Umweltdaten
- **Echtzeit-Daten** – AQI, Pollen (Birke, Gräser, Erle), UV-Index, PM2.5
- **Beste Tageszeit** – optimales Zeitfenster für Outdoor-Aktivitäten
- **Familienprofile** – individuelle Bewertung für bis zu 5 Personen
- **Offline-Modus** – letzte bekannte Daten bei fehlender Verbindung
- **Keine Registrierung** – datenschutzfreundlich, kein Tracking

## Datenschutz

Siehe [PRIVACY.md](PRIVACY.md) für die vollständige Datenschutzerklärung.

**Kurzfassung:** AirOut erhebt keine personenbezogenen Daten. Standortkoordinaten werden nur lokal verarbeitet. Keine Werbung, kein Tracking.

## Datenquellen

- [Open-Meteo](https://open-meteo.com) – kostenlose, offene Wetter- und Pollen-API
- [OpenStreetMap / Nominatim](https://nominatim.org) – Ortsnamen per Reverse Geocoding
